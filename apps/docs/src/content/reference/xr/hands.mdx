`<Hands />` creates XRHand inputs for devices that allow hand tracking.

This component can accept children in `left` and `right` slots, which will be parented to the hand of the respective handedness.

```svelte
<Hands>
  <T slot='left' is={$gltf1.scene} />
  <T slot='right' is={$gltf2.scene} />
</Hands>
```
