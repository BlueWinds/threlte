`<Controllers />` represents a THREE.XRTargetRaySpace, a THREE.XRGripSpace, and a controller model per each hand.

You can specify custom models, or no models to be rendered for these controllers.

This component can accept children in `left` and `right` slots, which will be parented to the controller of the respective handedness.

```svelte
<Controllers>
  <T slot='left' is={$gltf1.scene} />
  <T slot='right' is={$gltf2.scene} />
</Controllers>
```
