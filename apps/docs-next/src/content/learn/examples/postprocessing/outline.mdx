Implements the Outline postprocessing pass. [Vanilla threejs example here](https://threejs.org/examples/?q=outline#webgl_postprocessing_outline)
An outlined cube loops through a maze, and it's outline color changes when the object is hidden.

- Postprocessing is achieved via `CustomRenderer` component.
- Cube animation done with `svelte/motion`
